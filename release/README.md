# Spirecity MVP Release

Это компактная версия проекта, подготовленная для деплоя на сервер.

## Инструкция по запуску:

1. **Установите зависимости:**
   Выполните команду в корне этой папки:
   ```bash
   npm install
   ```

2. **Настройте окружение:**
   Переименуйте `server/.env.example` в `server/.env` и укажите свой `JWT_SECRET`.

3. **Сбросьте базу данных (опционально):**
   Если нужно очистить БД и создать админа (admin/admin123!):
   ```bash
   npm run reset-db
   ```

4. **Запустите сервер:**
   ```bash
   npm start
   ```

Приложение будет автоматически раздавать фронтенд и API на порту 3000 (или другом из .env).

# Spirecity MVP Release

Это компактная версия проекта, подготовленная для деплоя на сервер.

## Структура папки:
- `dist/` — Скомпилированный фронтенд (React).
- `server/` — Бекенд (Express + SQLite).

## Инструкция по запуску:

1. **Установите зависимости сервера:**
   Зайдите в папку `server` и выполните `npm install`.
   Или просто выполните `npm run install-all` в корне этой папки.

2. **Настройте окружение:**
   Переименуйте `server/.env.example` в `server/.env` и укажите свой `JWT_SECRET`.

3. **Сбросьте базу данных (опционально):**
   Если нужно очистить БД и создать админа (admin/admin123!):
   `npm run reset-db`

4. **Запустите сервер:**
   `npm start`

Приложение будет доступно на порту 3000 (или другом из .env).
